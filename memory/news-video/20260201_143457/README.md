# 新闻视频素材

## 文件说明
- `script.txt` - 视频脚本
- `subtitles.srt` - 字幕文件 (SRT格式)
- `audio_placeholder.wav` - 音频占位符 (需要TTS生成)

## 脚本内容摘要
1. Bitcoin 价格波动
2. Binance 10亿美元承诺
3. Trump 提名美联储主席
4. 中国太阳能领先
5. Colorado AI法案生效

## 下一步
需要安装以下工具完成视频制作：
- TTS: espeak / pico / ElevenLabs API
- 视频: ffmpeg / remotion / moviepy
