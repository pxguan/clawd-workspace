# Awesome OpenClaw Skills

获取于: https://github.com/VoltAgent/awesome-openclaw-skills

## 安装方式

### ClawdHub CLI
```bash
npx clawdhub@latest install <skill-slug>
```

### 手动安装
复制 skill 文件夹到：
- **全局:** `~/.openclaw/skills/`
- **工作区:** `<project>/skills/`

## 部分精选技能

### Web & Frontend Development
- **frontend-design** - 高质量前端界面设计
- **remotion-server** - 无头视频渲染（无需 Mac 或 GUI）
- **vercel-react-best-practices** - Vercel React 性能优化指南

### Git & GitHub
- **conventional-commits** - 使用 Conventional Commits 格式化提交信息
- **github** - 通过 gh CLI 与 GitHub 交互
- **github-pr** - 获取、预览、合并和测试 GitHub PR

### DevOps & Cloud
- **cloudflare** - 通过 Wrangler CLI 管理 Cloudflare Workers
- **coolify** - 通过 Coolify API 管理部署
- **digital-ocean** - 通过 DO API 管理 Droplets
- **hetzner** - Hetzner Cloud 服务器管理
- **proxmox** - Proxmox VE 集群管理

### Coding Agents
- **claude-team** - 通过 iTerm2 协调多个 Claude Code workers
- **cursor-agent** - 使用 Cursor CLI agent
- **coding-agent** - 运行 Codex CLI, Claude Code, OpenCode

### CLI Utilities
- **domain-dns-ops** - 跨 Cloudflare, DNSimple, Namecheap 的 DNS 操作
- **exa-plus** - 通过 Exa AI 进行神经网页搜索
